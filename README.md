## ✨ Features

- 🎨 **Glass Morphism Design** — Modern frosted glass UI aesthetic across all sections
- 🌀 **Scroll-Triggered Animations** — Staggered entrance animations using Framer Motion
- ✨ **Interactive Cursor Effect** — Custom sparkle cursor animation for enhanced UX
- 📱 **Fully Responsive** — Optimized across 3 breakpoints (480px / 768px / 1024px)
- ⚡ **Vite Build** — Zero-error CI/CD pipeline with automatic Vercel deployment on every push
- 🧩 **Component-Based Architecture** — Clean, reusable React components

## ⚙️ Getting Started

### Prerequisites
- Node.js v18+
- npm

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/HeenalMewada/portfolio.git
cd portfolio

# 2. Install dependencies
npm install

# 3. Start development server
npm run dev
```

App runs at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

## 🔧 Technical Highlight

Resolved a Vite 8 / `@vitejs/plugin-react` dependency conflict that was breaking the production build — resulting in a clean, zero-error CI/CD deployment pipeline on Vercel with automatic deployments triggered on every push to `main`.
